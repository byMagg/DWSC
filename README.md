# DWSC
El proyecto final se trata de una aplicación que hace uso de la API de Spotify, tenemos varios microservicios con SpringBoot los cuales de encargan de cada una de las funciones de:

- Añadir canción
- Verificar canción
- Eliminar canción
- Añadir comentario
- Eliminar comentario
